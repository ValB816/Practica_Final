# Practica-Final

-Descripción del proyecto

Este proyecto implementa un visualizador gráfico de ajedrez desarrollado en Java utilizando la librería Swing.
Permite ingresar una cadena FEN (Forsyth–Edwards Notation) para representar una posición de ajedrez en el tablero.
El sistema valida, interpreta y muestra gráficamente las piezas según la cadena ingresada.

-Uso de la aplicación

1.) Ingresa una cadena FEN en el campo de texto.
Ejemplo:
rnbqkbnr/pppppppp/8/8/8/8/PPPPPPPP/RNBQKBNR w KQkq - 0 1

2.) Presiona el botón “Validar y Mostrar” para generar el tablero.

3.)Las piezas se mostrarán según la posición indicada.

4.)Aparecerá el turno actual (blancas o negras).

5.)Si intentas repetir el mismo turno consecutivamente, el sistema mostrará una advertencia.

6.)Usa el botón “Limpiar tablero” para reiniciar la interfaz.

-Características de la interfaz

*Tablero 8x8 con colores tradicionales.

*Piezas en formato Unicode (♔ ♕ ♖ ♗ ♘ ♙).

*Coordenadas alrededor del tablero (A–H y 1–8).

*Etiqueta de turno con cambio de color dinámico:

*Fondo blanco con texto negro cuando juegan las blancas.

*Fondo gris oscuro con texto blanco cuando juegan las negras.

*Diseño adaptable al tamaño de la ventana.
